COMMENT IMPLEMENTER UN MENU DYNAMIQUE POUR LES FRAMES
Pourquoi ce menu est-il dynamique ?
Il est dynamique car il permet de faire dérouler et enrouler les options d'une rubrique.
Exemple de menu : 

RUBRIQUE n°1
	Option 1.1
	Option 1.2
	Option 1.3
RUBRIQUE n°2
	Option 2.1
	Option 2.2
.............
En cliquant sur une rubrique ouverte, elle se ferme. Et vice-versa.
Cette présentation du menu est très efficace quand le contenu est dense.
Vous trouverez dans la rubrique "Boutique" du site Tout JavaScript.com une extension de ce script permettant d'afficher des sous rubriques sans limitation de niveaux.
