Description

Ce script est une variante de Tableau avec colonnes multi-pages. La différence est que lorsqu'une requête est exécutée, le script parcourt les résultats, récupère les longueurs des chaînes et redimensionne les colonnes pour essayer de tout mettre dans la page. Si une colonne n'utilise pas toute la largeur qui lui est allouée, elle est réduite et l'espace récupéré réalloué à une autre colonne. Le but est d'obtenir une présentation agréable, mais l'inconvénient est que le temps d'exécution peut être un peu long. C'est pourquoi il y a une option de dump qui affiche du code PHP pour fixer la taille des colonnes, on peut ainsi faire un test avec une petite quantité de données, puis utiliser les valeurs obtenues sur l'ensemble.
